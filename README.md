# Estrutura-de-Dados-Avan-adas
Este repositório armazena atividades e exercícios práticos de Estrutura de Dados Avançados em C. Abrange algoritmos recursivos, árvores binárias, AVL e rubro-negras, além de teoria dos grafos e buscas (BFS, DFS, menor caminho), consolidando conceitos teóricos por meio de implementações.
